# java-calculator

A simple Java-based calculator capable of performing basic arithmetic operations such as addition, subtraction, multiplication, and division. Built using core Java concepts like conditional statements, methods, and user input through the Scanner class. Designed for beginners to practice logic building and syntax in Java.

# Features
- Performs addition, subtraction, multiplication, and division  
- Uses separate methods for each operation  
- Handles invalid input (e.g., divide by zero)  
- Beginner-friendly and easy to understand  
- Clean console-based interface  

# Technologies Used
- **Java (Core)**
- **Scanner class** for input  
- **Conditional statements**  
- **Methods** (functions in Java)

# How to Run the Program
1. Using Terminal / Command Prompt
   javac MethodCalculator.java
   java MethodCalculator

   
# **2. Using VS Code / IntelliJ / Eclipse**
- Open the folder  
- Run the `MethodCalculator` file  
- Enter numbers and choose an operation
